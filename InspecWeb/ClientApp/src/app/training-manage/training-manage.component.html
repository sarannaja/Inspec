การจัดการ
# เมนูการจัดการ ใช้งาน
1 อัพโหลดเอกสารเพิ่มเติม ( ทุกช่วงเวลา )
2 กำหนดคุณสมบัติ ( ช่วงการสร้างหลักสูตร )
3 ตรวจสอบคุณสมบัติผู้สมัคร ( หลังสิ้นสุดการสมัครอบรม )
4 กำหนดช่วงเวลาอบรม ( ก่อนเริ่มการอบรม )
5 ออกรหัสเข้าอบรม ( ก่อนเริ่มการอบรม )
6 บัตรผู้ร่วมอบรม ( หลังออกรหัสเข้าอบรม )
7 จัดกลุ่มผู้สมัคร ( ก่อนเริ่มการอบรม )
8 แบบประเมินหลักสูตรวิทยากร ( หลังการอบรมเสร็จสิ้น )
9 รายชื่อผู้เข้าร่วมอบรม ( หลังการอบรมเสร็จสิ้น )
10 รายงาน ( หลังการอบรมเสร็จสิ้น )
